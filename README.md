📚 StudyAI – Smart AI Study Companion & Quiz Generator

StudyAI is an AI-powered web application that helps students learn more efficiently by automatically generating study materials from uploaded documents. Users can upload PDF, DOCX, or TXT files, and the application uses AI to create concise summaries, revision flashcards, quizzes, and personalized study plans.

Built with a modern React frontend and Flask backend, StudyAI integrates the Groq AI API for fast content generation and provides a simple, user-friendly interface for smarter learning.

✨ Features
📄 Upload PDF, DOCX, and TXT study materials

🃏 Automatic flashcard generation
❓ AI-powered quiz generation

🚀 Workflow
Upload a study document.
Backend stores the file securely.
Text is extracted from the document.
Extracted text is sent to the Groq AI model.
AI generates summaries, flashcards, quizzes, or study plans.
Results are displayed instantly in the React frontend.

🎯 Objective

StudyAI aims to simplify learning by reducing the time students spend creating notes and revision material. Instead of manually preparing summaries or quizzes, students can focus on understanding concepts while AI generates personalized study resources automatically.
